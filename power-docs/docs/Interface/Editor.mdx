# Editor

The **Power Editor** is the main window you'll use while working with Power.

<img src={Editor} alt="Editor reference" />

Power is built around creating assets and connecting them together to create the game mechanics you need.

Although every Power asset can be created from the **Assets → Create** menu, the **Power Editor** makes it much easier to create, organize, and manage your Power assets.

You can open the **Power Editor** from **Power → Editor**.
