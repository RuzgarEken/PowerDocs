# Reflector

It allows reflection to be performed according to the incoming Effect value sign and redirects the final calculated Effect value to one or more Actors based on the configuration defined by the resolved **Reflector Stat** and its Signature.

Reflection is evaluated separately for the **Source Actor** and the **Target Actor**.

The Reflector Stat is resolved as follows:

- For the **Target Actor**, the Reflector Stat is obtained directly from the **Target Stat's Reflector Relative Stat**.
- For the **Source Actor**, the Reflector Stat is obtained from the **Source Actor** using the **Stat Key** referenced by the **Target Stat's Reflector Relative Key**.

The resolved Stat must be a **Reflector Stat**.
