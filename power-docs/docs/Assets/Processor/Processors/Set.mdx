# Set

The **Set** Processor calculates the Effect value according to the processed Effect's configuration and directly sets the target Stat's **RawValue** to the resulting value.

Unlike Modifier-based Processors, the **Set** Processor does not create a Modifier.

## Supported Types

- Actor
- Float
- Int
- EffectPack
- String
- Vector2
- Vector3
- Vector2Int
- Vector3Int
