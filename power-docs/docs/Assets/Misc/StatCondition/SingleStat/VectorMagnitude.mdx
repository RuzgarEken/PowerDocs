# Vector Magnitude

A **Vector Magnitude** checks whether the magnitude of a vector satisfies a comparison rule.

The vector's magnitude is evaluated using a **Numerable Comparer**.

## Fields

| Field | Description |
| --- | --- |
| **Comparer** | The **Numerable Comparer** used to evaluate the vector's magnitude. |

### Supported Types

- **Vector2**
- **Vector3**
- **Vector2Int**
- **Vector3Int**
