# Distance

A **Distance Condition** calculates the distance between two Vector Stats and evaluates the result using a comparison condition.

## Fields

| Field | Description |
| --- | --- |
| **Comparer** | The **Numerable Comparer** used to evaluate the calculated distance. |

### Supported Types

- **Vector2**
- **Vector3**
- **Vector2Int**
- **Vector3Int**
