# Difference

A **Difference Condition** subtracts one Stat value from another and evaluates the result using a comparison condition.

## Fields

| Field | Description |
| --- | --- |
| **Comparer** | The **Numerable Comparer** used to evaluate the difference between the two Stat values. |

### Supported Types

- **Float**
- **Int**
