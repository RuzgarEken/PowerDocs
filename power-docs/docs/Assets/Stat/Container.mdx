Each Container stores **Stat Definitions** of a specific type.

<img src={Container} alt="Container reference" />

For example:

- **IntContainer** stores `Int` Stat Definitions.
- **GeneratorFloatContainer** stores `GeneratorFloat` Stat Definitions.
- **StateBasedEffectContainer** stores `StateBasedEffect` Stat Definitions.

## Fields

| Field | Description |
| --- | --- |
| **Container Id** | The Container's unique identifier within the **Stat Environment**. **Stat Keys** use this Id to locate and access the Container. |
| **Stats** | The list of **Stat Definitions** contained in this Container. |
