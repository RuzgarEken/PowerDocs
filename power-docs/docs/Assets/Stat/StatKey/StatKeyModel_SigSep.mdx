# StatKeyModel_SigSep

A **StatKeyModel_SigSep** uniquely identifies a Stat stored inside a **SigSep Stat**.

## Fields

| Field | Description |
| --- | --- |
| **Environment** | The **Environment Id** of the Environment that contains the Stat. |
| **Container** | The **Container Id** of the Container that stores the Stat. |
| **Stat** | The **Stat Id** of the SigSep Stat. |
| **Inner Stat** | The **Stat Id** of the inner Stat stored within the SigSep Stat. |
