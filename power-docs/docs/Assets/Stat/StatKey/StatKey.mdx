# Stat Key Model

A **Stat Key Model** uniquely identifies the location of a Stat within a **Stat Repository**.

## Fields

| Field | Description |
| --- | --- |
| **Environment** | The **Environment Id** of the Stat. This specifies which Stat Environment contains the Stat. |
| **Container** | The **Container Id** of the Stat. This specifies which Stat Container stores the Stat. |
| **Stat** | The **Stat Id** of the Stat within its Container. |
