# Environment

A **Stat Environment** groups different types of **Stat Containers** together.

<img src={Environment} alt="Environment reference" />

## Fields

| Field | Description |
| --- | --- |
| **Parent Environment Id** | Environments can have a parent-child relationship within an Actor. When `GetStat` is called and the requested Stat cannot be found in the current Environment, the lookup continues in the parent Environment. This can be thought of as an inheritance mechanism, where one Environment inherits from another, overrides specific Stats, and introduces additional functionality. |
| **Stat Containers** | A list of **Stat Containers** of different types contained in this Environment. |
