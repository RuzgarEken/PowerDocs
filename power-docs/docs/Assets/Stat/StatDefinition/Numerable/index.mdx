# Numerable

A **Numerable Stat** is the base type for all numeric Stats.

It provides common functionality shared by numeric Stat types, such as arithmetic operations, comparisons, and value-based processing.

## Fields

This Stat inherits all fields from **[Stat Definition](../index.mdx)**.

| Field | Description |
| --- | --- |
| **Default** | The initial value assigned to the Stat when it is created. |
