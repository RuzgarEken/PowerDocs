# Actor

An **Actor Stat Effect** modifies **Actor Stats**.

It inherits the common Effect behavior from **Effect** and assigns an Actor reference to the target Stat.

## Fields

This Effect inherits all fields from **[Effect](./index)**.

| Field | Description |
| --- | --- |
| **Interaction Actor** | Determines which Interaction Actor is assigned to the Actor Stat. |
| **Custom Actor** | Specifies a custom Actor to assign when using a custom Interaction Actor. |
| **Actor Stat From** | Determines which Actor the Actor Stat is retrieved from when **Interaction Actor** is set to **Actor Stat**. |
| **Custom Actor For Actor Stat** | Specifies the custom Actor from which the Actor Stat is retrieved. |
| **Actor Stat Key** | The Stat Key used to retrieve the Actor Stat. |
