# Effect Pack

An **Effect Pack Stat Effect** modifies **Effect Pack Stats**.

It inherits the common Effect behavior from **Effect** and uses **Effect Packs** as the value applied by the Effect.

## Fields

This Effect inherits all fields from **[Effect](./)**.

| Field | Description |
| --- | --- |
| **Use Stat Value** | Uses a Stat instead of a predefined Effect value. |
| **Value** | The predefined Effect Pack applied by the Effect. |
| **Value Stat** | The Stat that provides the Effect Pack value. |
