# General Expansions

`General Expansions` contains systems and extensions that are not part of the Power Core but significantly extend Power's capabilities.

Since the full source code is included, you are free to modify and integrate these systems into your own projects as needed.

:::warning

Demo projects rely on the systems included in **General Expansions**. Always import **General Expansions** before importing any demo project to ensure all required assets and dependencies are available.

:::
