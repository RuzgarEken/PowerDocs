# Premade Asset Pack

`Premade Asset Pack` (PAP) is an editor asset designed to simplify the creation of frequently used assets that do not contain Stat relations.

Like `Premade Relation Pack` (PRP), PAPs help eliminate repetitive work by allowing you to create commonly used assets once and reuse them across projects.

Create the required assets and add them to the PAP as references.

<img src={PapReference} alt="Pap reference" />

## Extracting Assets

Use the **Extract** section to choose the destination folder where the referenced assets will be created.
