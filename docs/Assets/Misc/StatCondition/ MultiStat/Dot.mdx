# Dot

A **Dot Condition** calculates the dot product of two Vector Stats and evaluates the result using a comparison condition.

## Fields

| Field | Description |
| --- | --- |
| **Comparer** | The **Numerable Comparer** used to evaluate the calculated dot product. |

### Supported Types

- **Vector2**
- **Vector3**
- **Vector2Int**
- **Vector3Int**
