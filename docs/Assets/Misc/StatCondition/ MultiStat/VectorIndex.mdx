# Vector Index

A **Vector Index Condition** compares the values of the selected component from two Vector Stats using a **Numerable Comparer**.

## Fields

| Field | Description |
| --- | --- |
| **Comparing Index** | The vector component to compare. |
| **Comparer** | The **Numerable Comparer** used to evaluate the selected component values. |

### Comparing Index

- **0**: X
- **1**: Y
- **2**: Z

### Supported Types

- **Vector2**
- **Vector3**
- **Vector2Int**
- **Vector3Int**
