# Add

The **Add** Processor calculates the Effect value and adds the resulting value to the target Stat.

For **Float** and **Int** Effects, the calculation is performed according to the configuration of the **Numerable Stat Effect**.

For **Vector2**, **Vector3**, **Vector2Int**, and **Vector3Int** Effects, the calculation is performed according to the configuration of the **Vector Stat Effect**.

## Supported Types

- Float
- Int
- Pack
- Vector2
- Vector3
- Vector2Int
- Vector3Int
