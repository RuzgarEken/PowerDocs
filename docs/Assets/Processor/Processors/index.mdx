# Processors

The **Processor** stage is the main stage of the processing pipeline where Effects are applied to their target Stats.

Processors execute the Effect's core behavior. Depending on the Processor, this may involve calculating the value to apply, adding or removing Modifiers, managing Injectors, or performing other operations on the target Stats.

In short, this stage is responsible for executing the primary logic of the Effect.
