# Reflector

The **Reflector** Processor redirects the final calculated Effect value to one or more Actors according to the configuration defined by the resolved **Reflector Stat** and its Signature.

Reflection is evaluated separately for the **Source Actor** and the **Target Actor**.

The Reflector Stat is resolved as follows:

### Base Stat (BS) Processors

- For the **Target Actor**, the Reflector Stat is obtained directly from the **Target Stat's Reflector Relative Stat**.
- For the **Source Actor**, the Reflector Stat is obtained from the **Source Actor** using the **Stat Key** referenced by the **Target Stat's Reflector Relative Key**.

### Attribute Stat (AS) / Multi Attribute Stat (MAS) Processors

- For the **Target Actor**, the Reflector Stat is obtained from the **Target Actor** using the **Stat Key** referenced by the **Attribute Stat's Reflector Relative Key**.
- For the **Source Actor**, the Reflector Stat is obtained directly from the **Attribute Stat's Reflector Relative Stat**, since the Attribute Stat already belongs to the Source Actor.

The resolved Stat must be a **Reflector Stat**.
