# Cast Restrictor

The **Cast Restrictor** is the signed version of the **Preprocess Cast Restrictor** Processor.

It determines whether the **Source Actor** is allowed to apply an Effect with the incoming value sign to the target Stat.

For example:

- The Source cannot apply **positive** Effects to the **Health** Stat.

This is a **Source-side only** Processor.

The Cast Restrictor Stat is resolved from the **Source Actor** using the **Stat Key** referenced by the **Target Stat's Cast Restrictor Relative Key**.

The resolved Stat must be an **Int Stat**.
